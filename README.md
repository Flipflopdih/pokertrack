# PokerTrack

Multiplayer Texas Hold'em with GTO accuracy scoring and full session stats.

## Deploy to Railway (5 minutes)

1. Go to https://github.com/new and create a new repository called `pokertrack`
2. Upload these files to it (drag and drop on GitHub works)
3. Go to https://railway.app → New Project → Deploy from GitHub repo
4. Select your `pokertrack` repo
5. Railway auto-detects Node.js and deploys — done!

Your app will be live at something like `pokertrack-production.up.railway.app`

## Local development

```bash
npm install
npm start
```

Then open http://localhost:3000
